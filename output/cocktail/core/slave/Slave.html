Constructor detail
Slave()constructor
public function Slave(auto_load:Boolean = false, parallelized:Boolean = false)

Creates a new Slave.

Parameters
auto_load:Boolean (default = false) — If true all subsequent loading calls will start loading immediatelly, otherwise false you'll need to call the "load" method to start the loading process.
 
parallelized:Boolean (default = false) — If true all queue objects are started to load in the same time, otherwise false the loading process is sequentially managed, where the object is started only when the prev object is quite and done.
